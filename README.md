# 💙 Commit Message Rules 
- 가능한 50자를 넘지 않게, 간단하지만 명료한 형태로 작성 
- (Commit Message) : (수정/작업한 부분에 대한 설명)의 형태로 작성  
- **✍🏻Commit Message List**  

    - Docs : 문서 편집 및 작성 
    - Fix : 버그 수정
    - Feat : 새로운 기능 구현
    - Modify : 기능 수정
    - Refactor : 리팩토링, 코드 수정(기능이 동일할 경우)
